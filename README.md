# Miro's Spine Visualization

An interactive visualization of a dog's spine showing how different movements affect specific disc regions.

## 🚀 Live Demo

Visit: https://[your-username].github.io/miro-spine-viz/

## 🎨 Features

- Interactive spine visualization
- Movement comparison (walking, jumping, stairs, etc.)
- Highlighted affected disc regions (C4-C6, T12-T13, T13-L1)
- Educational information about each vertebral region
- Responsive design

## 🛠️ Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## 📦 Technologies

- React 18
- Vite
- SVG animations
- GitHub Pages

## 📝 About

This visualization was created to help understand the impact of different movements on Miro, a 4.5-year-old teacup Morkie with IVDD (Intervertebral Disc Disease).
